Contributing#

Do you have any interesting additions to the benchmark battery? Let’s prepare its Version 2.0 some time soon together!

Contributions of new datasets and/or label vectors following the File Format Specification are warmly welcome.

Please contact the maintainer/editor or post an issue on GitHub.

Researchers: Please test your new algorithms against all the datasets listed here and submit the raw results they generate.

Reviewers: Please ask the authors to verify their proposals against this benchmark suite.
